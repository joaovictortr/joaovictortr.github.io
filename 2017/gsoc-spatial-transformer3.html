GSoC - Spatial Transformer 3

// under python gsoc

Spatial transformer implementation in the GPU using cuDNN is now complete. The problem with the gradient computation was solved, with the help of my mentor. Also, comprehensive unit tests of the Op and the gradients were added. Further details can be found in the units tests, and discussion in the pull request.

As future direction, and last part of my GSoC project, I will implement a CPU version of the spatial transformer, using symbolic operations from Theano. The initial implementation and discussions can be followed on the appropriate pull request as well.
